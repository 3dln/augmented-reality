# Gif component
A component which renders an image on a flat plane.

## Markdown
```
<a-assets>
  <img id="image" src='./image.jpg' alt="" />
</a-assets>
<a-marker preset="hiro">
  <a-image
    width="4"
    height="4"
    position="0 0 0"
    src="#image"
  ></a-image>
</a-marker>
```
