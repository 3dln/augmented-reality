# Box component
A component which renders the box geometry component.
Other similar elements can be:
* Plane
* Circle
* Cone
Check the documentation for more shapes and geometries.

## Markdown
```
<a-box position="0 0 0" material="color: yellow; opacity: 0.5;"></a-box>
```
